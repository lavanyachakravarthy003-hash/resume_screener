# 🤖 AI Resume Screening System (ATS)

An AI-powered Applicant Tracking System (ATS) that automatically analyzes resumes, matches them with job descriptions, and ranks candidates based on relevance using NLP and machine learning techniques.

---

## 🚀 Live Demo
🔗 https://lavanyachakravarthy003-hash-resume-screene-streamlit-app-abxuzq.streamlit.app/

---

## 📌 Features

- 📄 **Dynamic Job Description Input**
  - Users can type/paste job descriptions directly

- 📂 **Multi Resume Upload**
  - Supports PDF and DOCX formats

- 🧠 **AI-Based Matching**
  - Uses NLP and semantic similarity to rank candidates

- 🛠 **Skill Extraction**
  - Automatically extracts skills from resumes and job descriptions

- ❌ **Skill Gap Analysis**
  - Identifies missing skills for each candidate

- 🎯 **Match Score**
  - Displays percentage-based ranking for easy evaluation

- 🎨 **Highlighted Skills**
  - Highlights matched skills in resumes

- 📊 **Analytics Dashboard**
  - Pie chart showing top skills distribution

- 📥 **Download Results**
  - Export shortlisted candidates as CSV

---

## 🛠️ Tech Stack

- **Frontend/UI:** Streamlit  
- **Backend:** Python  
- **NLP:** spaCy  
- **ML/Similarity:** Sentence Transformers  
- **Data Handling:** Pandas  
- **Visualization:** Matplotlib
